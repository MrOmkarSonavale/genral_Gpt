ChatGPT Clone – AI Powered Conversational Platform

This project is a full-stack AI chatbot application inspired by ChatGPT. It allows users to interact with an intelligent conversational agent capable of generating contextual responses using a large language model API.
The system is designed with a scalable architecture where the frontend handles the chat interface while the backend processes requests and communicates with the AI model API.

Core Features

Real-time conversational interface
AI response generation using LLM APIs
Chat history storage and retrieval
Clean and responsive UI
REST API-based backend architecture

Tech Stack
Frontend: React
Backend: Node.js, Express
Database: MongoDB
AI Integration: OpenAI API
